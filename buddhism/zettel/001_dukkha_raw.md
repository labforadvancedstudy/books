# Dukkha: The Prediction Error of Existence

## 정확한 이해: 둑카 ≠ 고통

**잘못된 번역의 문제**:
- "Life is suffering" → 염세주의로 오해
- "Everything is pain" → 허무주의로 왜곡
- 실제: "뭔가 맞지 않는 느낌" (unsatisfactoriness)

**신경과학적 정의**:
```python
dukkha = prediction_error + resistance_to_update
```

## 신경학적 메커니즘

### Anterior Cingulate Cortex (ACC)의 역할

**발견**: Prediction error의 신경 기질
- Eisenberger & Lieberman (2004): 사회적 거절 = 물리적 통증
- ACC 활성화 패턴 동일
- 진통제(아세트아미노펜)가 둘 다 완화

## 둑카의 3가지 층

### 1. Dukkha-dukkha (고고, 苦苦)
**명백한 고통**:
- 물리적: 질병, 부상, 노화
- 정서적: 슬픔, 분노, 두려움
- 사회적: 거절, 고립, 상실

**신경 마커**:
- Pain matrix activation
- Cortisol elevation
- Inflammatory cytokines ↑

### 2. Viparinama-dukkha (괴고, 壞苦)
**변화의 고통**:
```python
def impermanence_suffering():
    examples = [
        'Pleasure → ends → disappointment',
        'Success → fades → emptiness',
        'Youth → ages → loss',
        'Love → changes → grief'
    ]
    
    mechanism = 'Hedonic adaptation + Loss aversion'
    # We adapt to gains quickly but feel losses deeply
    return 'The suffering of all good things ending'
```

### 3. Sankhara-dukkha (행고, 行苦)
**조건지어진 존재의 고통**:
- 가장 미묘하고 깊은 층
- 매 순간 유지하려는 노력
- "자아"를 구성하는 에너지 소모


## 핵심 통찰

둑카는 버그가 아니라 기능입니다:
1. 생존을 위한 동기 부여 시스템
2. 학습과 적응의 신호
3. 현실과 기대의 차이 감지기

문제는 둑카 자체가 아니라:
- 과도한 저항 (suffering about suffering)
- 만성적 활성화 (현대 생활의 미스매치)
- 둑카와의 잘못된 관계

해결은 제거가 아닌 재구성:
- 둑카를 정보로 보기
- 저항 대신 탐구
- 고통에서 지혜로
