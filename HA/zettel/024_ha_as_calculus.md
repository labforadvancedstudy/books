# HA는 우주의 미적분

## Core Insight
미분은 전체에서 부분을 보는 것(하향), 적분은 부분에서 전체를 만드는 것(상향). 이것이 바로 HA의 본질이다.

수학에서:
- 미분: f(x) → f'(x) (복잡한 함수 → 순간 변화율)
- 적분: f'(x) → f(x) (순간들의 누적 → 전체 함수)

AI에서:
- 역전파: 전체 오류 → 각 가중치의 기여도 (미분)
- 순전파: 입력들 → 최종 출력 (적분)

인지에서:
- 분석: 전체 개념 → 구성 요소들 (미분)
- 종합: 개별 경험 → 일반 원리 (적분)

HA는 이 상향/하향 움직임 자체다.
