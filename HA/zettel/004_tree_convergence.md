# 트리 구조로의 수렴

## Core Insight
자연의 모든 지속 가능한 시스템은 트리 또는 계층적 구조로 수렴한다. 이는 정보와 에너지 전달의 최적 구조이기 때문이다.

자연의 예시:
- 하천: 작은 물줄기 → 지류 → 강 → 바다
- 혈관: 모세혈관 → 정맥 → 대정맥 → 심장
- 신경계: 말초신경 → 신경절 → 척수 → 뇌
- 진화: 종 → 속 → 과 → 목 → 강 → 문 → 계

트리는 n개 노드를 n-1개 연결로 완전히 연결한다. 최소 연결로 최대 도달성을 보장하는 구조다.
