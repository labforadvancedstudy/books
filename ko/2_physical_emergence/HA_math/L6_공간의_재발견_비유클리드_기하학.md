> "진리란 무엇인가? 단지 오래된 편견은 아닐까?"

## 2000년의 집착

### 알렉산드리아, 기원전 300년

유클리드가 『원론』을 완성합니다. 13권의 책, 465개의 명제. 

다섯 개의 공리에서 출발:
1. 두 점을 지나는 직선은 하나
2. 선분은 양방향으로 연장 가능
3. 한 점에서 같은 거리의 점들은 원
4. 모든 직각은 같다
5. **평행선 공리** (복잡한 버전)

처음 네 개는 간단명료. 다섯 번째는...

"직선 밖의 한 점을 지나 그 직선과 만나지 않는 직선은 오직 하나뿐이다."

뭔가 이상합니다. 너무 복잡하고, 덜 자명해 보입니다.

프로클로스(5세기): "이건 증명할 수 있을 거야!"

## 쌍곡 기하학: 안장 위의 세계

### 로바체프스키-볼리아이 기하학

평평한 종이가 아닌 말안장 모양의 공간을 상상하세요.

```
     안장 표면
    __~~__
   /      \
  (        )
   \__  __/
      \/
```

이 공간에서는:
- 평행선이 무수히 많다
- 삼각형 내각의 합 < 180°
- 원주율 > π (원이 더 크다!)

### 구체적 모델: 푸앵카레 원판

앙리 푸앵카레가 만든 모델:

```
      경계 (무한히 멀다)
    /                   \
   |    ___     ___      |
   |   /   \   /   \     |  <- "직선들"
   |  |     \ /     |    |
   |   \     X     /     |
    \   \__/   \__/     /
      ---------------
```

원의 경계는 무한히 멀고, 가운데서 가장자리로 갈수록 거리가 늘어납니다.
에셔의 "Circle Limit" 작품들이 바로 이것!

## 리만의 도약: 구면 기하학

### 괴팅겐, 1854년

베른하르트 리만의 취임 강연. 가우스가 청중석에 있습니다.

"여러분, 더 과감해집시다. 평행선이 **아예 없다면**?"

지구 표면을 생각하세요:
- "직선" = 대원 (최단 경로)
- 모든 "직선"은 만난다 (남극과 북극에서)
- 삼각형 내각의 합 > 180°

```
   북극
    /|\
   / | \    <- 경도선들 (모두 "직선")
  /  |  \
 /   |   \
---------- 적도
```

### 일반화: 리만 다양체

리만은 더 나아갑니다:

"공간의 곡률이 점마다 다르다면?"

```
평평한 공간: 곡률 = 0 (유클리드)
구: 곡률 > 0 (양의 곡률)
안장: 곡률 < 0 (음의 곡률)
일반: 곡률이 변한다!
```

이것이 60년 후 아인슈타인의 손에서...

## 기하학의 동물원

### 분류

| 기하학 | 평행선 | 삼각형 내각 합 | 곡률 | 모델 |
|--------|--------|----------------|------|------|
| 유클리드 | 1개 | 180° | 0 | 평면 |
| 쌍곡 | 무수히 많음 | < 180° | 음 | 안장 |
| 타원 | 0개 | > 180° | 양 | 구면 |

### 실생활 예시

**유클리드**: 책상 위
- 평행선은 만나지 않는다
- 피타고라스 정리 성립

**구면**: 지구 표면
- 서울 → 뉴욕 최단 경로는 곡선(대원)
- GPS가 이를 계산

**쌍곡**: ???
- 자연에서 찾기 어렵다
- 하지만 상대성 이론의 시공간이...

## 아름다운 실수들

칸트도 틀렸고, 뉴턴도 틀렸다.

### 2. "비유클리드 기하학은 쓸모없다"

오늘날 구글 맵이 작동하는 이유!

### 3. "기하학은 하나뿐이다"
아니다. 무수히 많다!

일정 곡률 기하학만 해도:
- 유클리드 (곡률 0)
- 구면 (곡률 k>0, 모든 k에 대해 다름)
- 쌍곡 (곡률 k<0, 모든 k에 대해 다름)

## 진짜 미스터리는...

**왜 우주는 거의 평평할까?**

관측 결과: Ωₖ = 0.001 ± 0.002

엄청나게 정확히 0에 가깝습니다. 왜?

- 우연의 일치?
- 인류 원리? (다르면 우리가 없다)
- 인플레이션? (초기 우주가 엄청나게 팽창)

더 깊은 질문들:
- 공간의 차원은 왜 3일까?
- 다른 우주는 다른 기하학을 가질까?
- 의식은 어떤 기하학 공간에 있을까?

리만: "공간의 본질은 여전히 어둠 속에 있다."

150년이 지난 지금도 마찬가지입니다.
