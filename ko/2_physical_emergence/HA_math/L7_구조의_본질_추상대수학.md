## 5차 방정식의 미스터리

### 성공의 역사

2차 방정식: 바빌로니아인들도 알았던 근의 공식
```
x = (-b ± √(b²-4ac))/2a
```

3차 방정식: 타르탈리아-카르다노 공식 (1545년)
복잡하지만 ∛를 사용해서 해결

4차 방정식: 페라리의 방법 (1545년)
3차로 변환해서 해결

그럼 5차는?

### 300년의 실패

1770년대 - 라그랑주: "뭔가 근본적으로 다르다..."
1799년 - 루피니: "불가능할 것 같은데..." (불완전한 증명)
1824년 - 아벨: "5차 이상은 일반적 근의 공식이 없다!"

하지만 왜? 

### 갈루아의 통찰

"해를 구하는 게 아니라, 해들의 관계를 봐야 한다!"

예시: x³ - 2 = 0의 해들
```
α = ∛2
β = ∛2 · ω  (ω = e^(2πi/3))
γ = ∛2 · ω²
```

이 해들을 섞는 방법:
- (α, β, γ) → (α, β, γ): 그대로
- (α, β, γ) → (α, γ, β): β와 γ 교환
- ... 총 6가지

**이 "섞는 방법들"이 군(group)을 이룬다!**

## 군: 대칭성의 대수학

### 군이란?

집합 G와 연산 ·이 다음을 만족:
1. **결합법칙**: (a·b)·c = a·(b·c)
2. **항등원**: e·a = a·e = a인 e 존재
3. **역원**: a·a⁻¹ = a⁻¹·a = e인 a⁻¹ 존재

"그게 다야?" 

네! 하지만 이 단순한 규칙에서 우주가 펼쳐집니다.

### 일상 속의 군

**시계 산술**
```
3시 + 10시간 = 1시
```
{0, 1, 2, ..., 11}과 +₁₂가 군!

**정사각형의 대칭**
```
  회전 90°     회전 180°    회전 270°
  ┌─┐         ┌─┐         ┌─┐
  │ │ →90°→  │ │ →90°→  │ │
  └─┘         └─┘         └─┘
```
8개의 대칭 (회전 4개 + 뒤집기 4개)

**루빅 큐브**
각 움직임이 원소, 연속 동작이 연산
4.3 × 10¹⁹개의 원소를 가진 군!

### 갈루아의 정리

**방정식이 근의 공식으로 풀린다 ⟺ 갈루아 군이 가해군이다**

가해군(solvable group): 점점 간단해지는 계열을 가진 군

5차 일반 방정식의 갈루아 군 = S₅ (120개 원소)
S₅는 가해군이 아니다!
따라서 근의 공식 없음!

## 추상화의 폭발

### 케일리: 순수 추상으로

아서 케일리 (1854년):
"군은 구체적 대상일 필요가 없다. 추상적 기호와 연산 규칙만 있으면 된다."

**곱셈표만 있으면 된다!**

```
크기 4인 군의 예:
  * | e a b c
  -----------
  e | e a b c
  a | a e c b
  b | b c e a
  c | c b a e
```

이게 무엇을 나타내든 상관없다!

### 클라인의 에를랑겐 프로그램 (1872년)

"기하학이란 무엇인가? 변환군에 대해 불변인 성질을 연구하는 것이다!"

- 유클리드 기하학: 회전, 평행이동, 반사에 불변
- 아핀 기하학: + 크기 변환에 불변
- 사영 기하학: + 투영에 불변
- 위상수학: 연속 변환에 불변

**기하학 = 군론!**

## 구조의 동물원

### 환(Ring)

덧셈과 곱셈이 있는 구조:
- 정수 ℤ
- 다항식 R[x]
- 행렬 M_n(R)

### 체(Field)

나눗셈도 되는 환:
- 유리수 ℚ
- 실수 ℝ
- 복소수 ℂ
- 유한체 F_p

### 벡터공간

체 위의 가군:
- 3차원 공간 ℝ³
- 함수 공간 C[0,1]
- 양자 상태 공간

## 에미 뇌터: 대칭성과 보존법칙

### 1915년, 괴팅겐

다비드 힐베르트가 에미 뇌터를 초청합니다.
"아인슈타인의 일반 상대성 이론에 문제가 있소. 에너지 보존이..."

교수회의 반대: "여자가 어떻게 교수가 되나?"
힐베르트: "우리는 대학이지 목욕탕이 아니오!"

### 뇌터의 정리 (1918년)

**"모든 연속 대칭성은 보존량을 낳는다"**

- 시간 대칭성 → 에너지 보존
- 공간 대칭성 → 운동량 보존
- 회전 대칭성 → 각운동량 보존

아인슈타인: "뇌터 양은 내가 아는 가장 중요한 수학적 천재다."

### 현대 물리학의 언어

표준 모델:
- SU(3) × SU(2) × U(1)
- 각 군이 기본 힘을 기술

대통일 이론 후보:
- E₈ × E₈
- 248차원의 리 군!
